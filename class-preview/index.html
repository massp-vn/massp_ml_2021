    Hello

    I'm Nhung Le, and I'll be the instructor for the DS-GA 1003 course this year. This is a temporary page to give you some information before the course begins (in January 2017). This year's class will be very similar to last year's class, though some of the more advanced topics may change.

    Prerequisites

    • Programming: You'll need to be comfortable with Python and NumPy, at least. Knowing how to generate plots and tables to summarize results will be necessary as well. You should take a look at the first homework assignment from a previous year to see what you'll be jumping into. For a crash course in Python for data science, the first few chapters of Data Science from Scratch seem like a good bet. For more in depth coverage, including the pandas library, Python for Data Analysis is worth a look. While pandas won't be particularly useful for the homework assignments, it's highly recommended for doing basic data analysis in practice and may be useful for your course projects. (If you're an R user, I highly recommend the data.table package.)
    • Math: Your best bet would be to review the notes from the prerequisite class DS-GA 1002. The priorities would be a review of matrix algebra (be comfortable with reading and manipulating expressions involving matrices, vectors, and norms), gradients, and basic probability theory (expectations, independence, Law of Large Numbers, conditional distributions, and conditional expectations). You'll also need to be comfortable with some basic analysis, such as taking limits and computing derivatives from first principles. But I consider pretty much every part of the 1002 syllabus (from fall 2015 -- this year it is split into two courses) to be an important part of a data scientist's toolbox.
    • Machine Learning: You should at least be familiar with the basic notions of supervised learning, overfitting, training set, test set, and cross-validation. The course is designed to follow DS-GA 1001, and many of your peers will have taken this class. The book Data Science for Business is highly recommended. It covers many important issues in practical data science that we don't have time for in this course. You might also consider working through Andrew Ng's Machine Learning course on Coursera. Essentially any introduction to machine learning would be sufficient.

    Textbooks

    The following 3 textbooks will serve as good references at different times during our course:

    • The Element of Statistical Learning (Hastie, Tibshirani, Friedman)
    • Understanding Machine Learning: From Theory to Algorithms (Shalev-Shwartz, Ben-David)
    • Pattern Recognition and Machine Learning (Bishop)

    The first two books are available as PDFs for free (legally). As a free alternative to Bishop's book above, you might consider the following:

    • Bayesian Reasoning and Machine Learning (David Barber)

    General Advice

    • This year's course lectures and homeworks will be similar to last year's. If you want to hit the ground running, it would not be a waste of time to start working on last year's homework assignments now.
    • The homework writeups must all be submitted as PDF or HTML files. There are many ways to do this, but it might be worth your time to figure out a way you're comfortable with before the class starts. For parts of the homework that are math heavy, you may want to use LyX or write directly in LaTeX. You can also write math directly in iPython, which can be convenient as well.
